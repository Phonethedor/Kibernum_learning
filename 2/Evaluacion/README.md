# Curriculum Vitae - Alvaro Castillo

Este proyecto consiste en un sitio web de portafolio personal / Curriculum Vitae digital, diseñado para presentar de manera profesional la experiencia laboral, educación y habilidades técnicas de Alvaro Castillo.

El sitio es totalmente responsivo y cuenta con una interfaz moderna que incluye un modo oscuro para mejorar la experiencia de usuario.

## Características Principales

*   **Diseño Responsivo:** Adaptable a dispositivos móviles, tablets y escritorio gracias a Bootstrap 5.
*   **Modo Oscuro (Dark Mode):** Interruptor funcional que permite alternar entre tema claro y oscuro, con persistencia de preferencia (si se implementa en JS, o al menos el switch está presente).
*   **Navegación Fluida:** Menú de navegación fijo en la parte superior con enlaces de desplazamiento suave a las diferentes secciones.
*   **Secciones Animadas:** Efectos de aparición (fade-in) al desplazarse por las secciones de experiencia y educación.
*   **Presentación Visual:** Uso de iconos FontAwesome y tipografías modernas (Poppins y Roboto) para una estética limpia y legible.

## Tecnologías Utilizadas

*   **HTML5:** Estructura semántica del contenido.
*   **CSS3:** Estilos personalizados, transiciones y variables para el manejo de temas.
*   **Bootstrap 5:** Framework CSS para la maquetación responsiva y componentes de interfaz (Navbar, Badges, etc.).
*   **JavaScript (ES6) y jQuery:** Lógica para interactividad, incluyendo el cambio de tema (modo oscuro) y animaciones al hacer scroll.
*   **FontAwesome:** Iconografía vectorial.
*   **Google Fonts:** Tipografías personalizadas.

## Estructura del Proyecto

*   `index.html`: Archivo principal con la estructura del sitio.
*   `assets/css/style.css`: Estilos personalizados y reglas para el modo oscuro.
*   `assets/js/script.js`: Scripts para interactividad y efectos.
*   `assets/icon/`: Contiene el favicon del sitio.
